### Endpoints

#### /users

- POST */users/signup* 
- POST */users/login* 
- POST */users/logout* 
- GET */users/account* 
- POST */users/resetToken* 
- POST */users/resetPassword*

- DELETE */pages/:pageId*
- POST */images*
- DELETE */images/:imageName*
